## 1. نظرة عامة على المشروع
**الهدف:** إنشاء متجر إلكتروني فاخر وتفاعلي لبيع العطور، يدعم اللغتين العربية والإنجليزية (واجهة عربية مع أسماء عطور إنجليزية)، مع توفير تجربة مستخدم سلسة تتضمن الفلترة، سلة التسوق، قائمة الأمنيات، والوضع الليلي.

**التقنيات المستخدمة:**
- **HTML5:** لبناء هيكل الصفحات الدلالي.
- **Vanilla CSS3:** لتنسيق الموقع بالكامل، استخدام المتغيرات (Variables)، الانتقالات (Transitions)، وتصميم متجاوب (Responsive Design).
- **Vanilla JavaScript:** لإدارة الحالة (State Management)، التفاعل مع المستخدم، والتخزين المحلي (localStorage).

## 2. خطوات التنفيذ (ماذا عملنا؟)

خلال جلسات العمل، تم تحويل المشروع من مجرد فكرة إلى متجر متكامل عبر الخطوات التالية:

1. **بناء الهيكل الأساسي:** إنشاء صفحات `perfurmes.html` (الرئيسية) و `shop.html` (المتجر).
2. **تصميم واجهة المستخدم (UI):** بناء أسلوب "Glassmorphism" مع ألوان فخمة (الذهبي، البيج، والأسود الفحمي).
3. **تطوير سلة التسوق وقائمة الأمنيات:** برمجتها بـ JS لتفتح كشريط جانبي (Sidebar) وتخزن البيانات محلياً.
4. **نظام الفلترة والبحث:** إضافة شريط بحث في الهيدر، وفلتر جانبي في صفحة المتجر لتصنيف العطور حسب الفئة والنوتة والسعر.
5. **الوضع الليلي الشامل (Dark Mode):** تعديل ملف CSS لإضافة متغيرات الوضع الداكن لجميع أقسام الموقع وتفعيل التبديل من الهيدر.
6. **الصور:** توليد صور عطور عالية الدقة بالذكاء الاصطناعي وربطها محلياً في مجلد `img` لضمان سرعة الظهور والموثوقية.
7. **إضافة أقسام تفاعلية:**
   - **الـ Hero:** خلفية سينمائية مع نص تفاعلي.
   - **عروض الموسم:** عداد تنازلي للعروض الخاصة.
   - **المدونة:** مقالات عن العطور لزيادة التفاعل.
   - **إتمام الشراء:** نافذة منبثقة (Modal) لمحاكاة عملية الدفع.
8. **التعريب:** ترجمة النصوص بالكامل للغة العربية مع الحفاظ على الرونق العالمي للأسماء الإنجليزية للمنتجات.
9. **التحسينات النهائية:** إصلاح مشاكل الـ Overlay (الطبقة الشفافة)، تباين الألوان في الوضع الليلي، وتجاوب الموقع مع شاشات الجوال.

## 3. هيكل الملفات
يتكون المشروع من الملفات التالية:

- `perfurmes.html` : الصفحة الرئيسية (تحتوي الهيدر، الـ Hero، المجموعات، العروض، المدونة، الفوتر).
- `shop.html` : صفحة المتجر المخصصة (تحتوي نظام الفلترة المتقدم والشبكة الكاملة للمنتجات).
- `success.html` : صفحة بسيطة تظهر بعد نجاح عملية الدفع.
- `styles.css` : ملف التنسيق الموحد لجميع الصفحات.
- `script.js` : ملف الجافاسكريبت الموحد الذي يحتوي على قاعدة بيانات المنتجات (Array) وكل وظائف التفاعل.
- `img/` : مجلد يحتوي على كافة صور العطور والخلفيات المستخدمة.

## 4. نظام التصميم (CSS & UI)

تم الاعتماد على متغيرات CSS (`:root`) لسهولة التحكم في الألوان وتطبيق الوضع الليلي:

### المتغيرات الأساسية (الوضع النهاري):
```css
:root {
  --beige: #f5f2e9;        /* خلفية رئيسية */
  --charcoal: #2d2d2d;     /* نصوص وعناصر داكنة */
  --gold: #d4af37;         /* اللون الذهبي المميز */
  --bg-color: var(--beige);
  --text-color: var(--charcoal);
  --card-bg: white;
}
```

### المتغيرات (الوضع الليلي):
```css
[data-theme="dark"] {
  --beige: #1a1a1a;
  --charcoal: #ffffff;
  --bg-color: #0d0d0d;
  --text-color: #e0e0e0;
  --card-bg: #1a1a1a;
}
```

### التصميم المتجاوب (Responsive):
تم استخدام `Flexbox` و `Grid` بكثافة. على سبيل المثال، شبكة المنتجات تستخدم:
`grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));`
مما يجعلها تتكيف تلقائياً مع حجم الشاشة (جوال، تابلت، كمبيوتر).

## 5. المنطق البرمجي (JavaScript)

تم تقسيم `script.js` إلى عدة أقسام وظيفية:

1. **قاعدة بيانات المنتجات (Products Data):** مصفوفة (Array) تحتوي على 11 عطراً مع تفاصيلها (الاسم، السعر، الفئة، النوتة، قوة التركيز، الصورة، والتقييم).
2. **إدارة الحالة (State Management):**
   - قراءة وكتابة السلة والأمنيات وحالة الوضع الليلي من `localStorage`.
3. **التصيير (Rendering):**
   - دالة `renderProducts()` تأخذ المنتجات وتقوم بتوليد كود HTML لكل بطاقة عطر وإضافته للصفحة.
4. **الفلترة (Filtering):**
   - دالة `filterProducts()` تقرأ حالة مربعات الاختيار (Checkboxes) ونطاق السعر (Range) وتقوم بتصفية مصفوفة المنتجات وإعادة تصييرها.
5. **سلة التسوق والأمنيات:**
   - دوال مثل `addToCart`, `removeFromCart`, `updateQuantity`, و `toggleWishlist`.
6. **التحكم في النوافذ (UI Toggle):**
   - فتح وإغلاق النوافذ الجانبية للسلة والطبقة المحجوبة `overlay`.

## 6. أكواد أساسية للإشارة

### هيكل الكارد الخاص بالعطر (مولد بواسطة JS):
```javascript
<div class="perfume-card reveal active">
  <button class="wishlist-btn" onclick="toggleWishlist(${product.id}, this)">
    <i class="far fa-heart"></i>
  </button>
  <div class="card-image">
    <img src="${product.image}" alt="${product.name}" />
  </div>
  <div class="perfume-info">
    <div class="rating">★★★★★ <span>(4.8)</span></div>
    <p class="category-tag">${product.notes} | ${product.category}</p>
    <h3>${product.name}</h3>
    <div class="price">$${product.price}</div>
    <button class="add-to-cart" onclick="addToCart(${product.id})">أضف إلى السلة</button>
  </div>
</div>
```

### دالة تطبيق الوضع الليلي:
```javascript
function toggleTheme() {
  currentTheme = currentTheme === 'light' ? 'dark' : 'light';
  document.documentElement.setAttribute('data-theme', currentTheme);
  localStorage.setItem('luxure_theme', currentTheme);
  updateThemeIcon();
}
```
